# cbp-ports
CBP JSON Data Ports

## Installation

Install via [npm](https://www.npmjs.com/):

```
$ npm install cbp-ports
```

## Methods

### getByUnloc(unloc)

Return the matched port, else undefined (alias: getByUNLOC, findByUnloc, findByUNLOC)
